Hot Springs

Problem Statement

There are $n$ pools of temperatures $t_1,\dots,t_n$, and we want to order them so the temperature between subsequent pools is increasing, i.e that:

$$|t_{i-1}-t_i|\leq |t_i-t_{i+1}|$$

Input

  • Number of pools $n$.
  • Random ordering of temperatures $t_i,...,t_n$

Output

The ordering of the temperature sequence that satisfies the requirement.

Reference Code (Python)


n = int(input())
a = list(map(int,input().split()))

a.sort()

i = n//2;

ans = []
for k in range(n):
    ans.append(a[i])
    if k & 1:
        i += k+1
    else:
        i -= k+1

print(" ".join(map(str,ans)))

Solution Explanation

We first sort out list of temperatures. We start in the middle of the array and add that first to our final ans array. The idea is to zig zag back and forth and add the values closer to the middle first (as they will have a smaller absolute difference), then work to the more extreme values (positive/negative or larger/smaller) and the ends of the sorted array. This will ensure their absolute difference is the largest combination in the sorted array.
