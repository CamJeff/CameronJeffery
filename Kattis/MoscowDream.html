Moscow Dream

Problem Statement

You are given an input for four numbers. The first is the number of easy problems to be on a test, the second is the number of medium problems, and the third is the number of hard problems. The last number is the number of questions on the test. If you can have at least 1 question of each difficulty on the test, print YES. If not, print NO.

Input

  • Four numbers. The first three are the number of easy, medium, and hard problem respectively. The fourth is the number of questions on the test.

Output

Whether its possible to have a test with at least question of each difficulty. YES if it is possible, or NO if not.

Reference Code (Python)


def main():
    easy, medium, hard, n = map(int, input().split())
    ok = (easy > 0 and medium > 0 and hard > 0 and n >= 3 and n <= easy + medium + hard)
    print("YES" if ok else "NO")

if __name__ == "__main__":
    main()

Solution Explanation

Save the number of easy, medium, hard problems and the number of test problems into four variables. We print YES if there are a non-zero amount of easy, medium, hard problems, and that the number of test problems is greater than 3, and is also less than or equal to the sum of easy, medium, hard problems (otherwise there would be questions on the test we couldn't fill).
