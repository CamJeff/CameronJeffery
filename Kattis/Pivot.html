Pivot

Problem Statement

Given an array, print out the number of pivots we could use for quicksort. A pivot means all elements to the left of it are smaller, and all elements to the right of it are larger.

Input

  • A number $n$ indicating the number of elements in the list.
  • A list of $n$ numbers separated by spaces

Output

The number of pivots that can be used.

Reference Code (Python)


def main():
    n = int(input())
    a = list(map(int, input().split()))
    INF = 10**19

    left_max = [None]*n
    m = -INF
    for i in range(n):
        left_max[i] = m
        if a[i] > m: m = a[i]

    right_min = [None]*n
    m = INF
    for i in range(n-1, -1, -1):
        right_min[i] = m
        if a[i] < m: m = a[i]

    print(sum(1 for i in range(n) if left_max[i] < a[i] < right_min[i]))

if __name__ == "__main__":
    main()

Solution Explanation

Get $n$, the number of elements, and save the numbers in a list/array called a. Set an arbitrarily large number to be used to compare min/max.

Create a left max array. For every number in our array, move leftwards until you find the maximum number. Save this maximum in variable m and the max array.

Similarly with the right side, you move backwards until you encounter the smallest element. Save these in the right min array.

Iterating through all numbers, print the number of elements where the left maximum at smaller than itself, and the right minimum is larger than itself. Increment by $1$ each time this happens.
