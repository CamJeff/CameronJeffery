Trik

Problem Statement

You have one ball and three opaque cups. You put the ball under the leftmost cup. Then you can make some shuffle moves. A: swap left and middle cup, B: swap right and middle cup, and C: swap left and right cup. Given a string sequence of these moves, print out which cup the ball is under. Print 1 for left cup, 2 for middle, 3 for right.

Input

  • Some string sequence of shuffle moves (using moves A, B, and C).

Output

The cup which contains the ball.

Reference Code (Python)


def main():
    moves = input().strip()
    pos = 1  

    for ch in moves:
        if ch == 'A':          # swap left & middle
            if pos == 1: pos = 2
            elif pos == 2: pos = 1
        elif ch == 'B':        # swap middle & right
            if pos == 2: pos = 3
            elif pos == 3: pos = 2
        else:                  # 'C' -> swap left & right
            if pos == 1: pos = 3
            elif pos == 3: pos = 1

    print(pos)

if __name__ == "__main__":
    main()

Solution Explanation

Save the list of shuffles in a string called moves. Initialize the variable pos to indicate the location of the ball. It starts under the left cup (1). Iterate through the move string, and update the pos of the ball accordingly based on the shuffle letter you see.
