Primary Register

Problem Statement

Given 8 numbers that are all written in modulo of the corresponding first 8 prime numbers. We increment the first number mod 2 by $1$. If it "overflows", we set it to 0, and increment the second number mod 3 by $1$. If that overflows, we set it to zero, and continue incrementing the third number mod 5. If the eighth number (mod 19) overflows, the computer blows up. Given a state of these 8 numbers, we want to determine how many times we can increment before the computer blows up.

Input

  • 8 numbers, $v_2,v_3,v_5,\dots,v_{19}$ that are each written modulo of the first 8 prime numbers.

Output

The maximum number of increments before the computer blows up.

Reference Code (Python)


def main():
    radices = [2, 3, 5, 7, 11, 13, 17, 19]
    v = list(map(int, input().split())) 

    value = 0     
    place = 1   
    for digit, base in zip(v, radices):
        value += digit * place
        place *= base      

    print(place - 1 - value)

if __name__ == "__main__":
    main()

Solution Explanation

The maximum number of increments we can do before blowing up: $2\times 3\times 5\times 7\times \cdots \times 19-1$. We save this in variable place, which is the maximum number of increments that we can get for each 'base' of modulo. For instance, if we have just two numbers $v_2$ and $v_3$, the maximum number of increments would be $1\times 2\times 3=6$ (we start with place=1).

The variable value represents the total number of increments that are happening, counting as they spill over on the other numbers. Here, it will always be equal to $v_x\times$place for the previous modulo base. For instance, if $v_5=4$, then value=$4\times (1\times 2\times 3)=24$. Of course, this is assuming that $v_5$ is the last number to spill over to. Thus, the number of increments possible before blowing up would be $(1\times 2\times 3\times 5)-24=6$, assuming $v_2=v_3=0$. If $v_2$ or $v_3$ were not $0$, then we would have to compute value for them, and add them to $24$, then subtract this from $30$.

This basic concpet is extended for all eight numbers.
