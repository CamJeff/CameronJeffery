Simon Says

Problem Statement

You are given a series of commands, only print out the commands that start with "Simon says". The output should have a space before the command.

Input

  • Number of commands $N$.
  • A list of $N$ commands, some start with "Simon says", some don't

Output

The commands that start with "Simon says".

Reference Code (Python)


n = int(input())

for i in range(n):
    line = input().split()
    if line[0] == "Simon" and line[1] == "says":
        print(" " + " ".join(line[2:]))

Solution Explanation

Go through each line, split line into array, if the first two elements are "Simon" and "says", print out the rest of the elements in that array.
