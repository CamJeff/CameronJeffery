Liga

Problem Statement

You are given a scoreboard of teams with 5 field from left to right: games played, games that won, drew, and loss, and the total number of points. You earn 3 points for a win, and 1 for a draw. Some fields in the scoreboard might be missing. Write a program that prints out the complete scoreboard.

Input

  • An integer $N$, indicated number of teams on the scoreboard.
  • Next $N$ lines are lines that have each of the five fields for the $N$ teams, but some fields may be missing, denoted as "?".

Output

The completed scoreboard.

Reference Code (Python)


for _ in range(int(input())):
    g, w, d, l, p = input().strip().split()
    unknowns = 5
    if g != '?': unknowns -= 1; g = int(g)
    if w != '?': unknowns -= 1; w = int(w)
    if d != '?': unknowns -= 1; d = int(d)
    if l != '?': unknowns -= 1; l = int(l)
    if p != '?': unknowns -= 1; p = int(p)

    if not unknowns:
        print(g, w, d, l, p)
    else:
        if g == '?' and l == '?':
            if w != '?' and d != '?': pass
            elif w != '?': d = p-3*w if p != '?' else 0
            elif d != '?': w = (p-d)//3 if p != '?' else 0
            else:
                for wd in range(10404):
                    w, d = wd//102, wd%102
                    if w+d < 101 and p==3*w+d: break
            print(100, w, d, 0, 3*w+d)

        elif g == '?':
            if w != '?' and d != '?': pass
            elif w != '?': d = p-3*w if p != '?' else 0
            elif d != '?': w = (p-d)//3 if p != '?' else 0
            else:
                for wd in range(10404):
                    w, d = wd//102, wd%102
                    if w+d+l < 101 and (p=='?' or p==3*w+d): break
            print(w+d+l, w, d, l, 3*w+d)

        elif l == '?':
            if w != '?' and d != '?': pass
            elif w != '?': d = p-3*w if p != '?' else 0
            elif d != '?': w = (p-d)//3 if p != '?' else 0
            else:
                for wd in range(10404):
                    w, d = wd//102, wd%102
                    if w+d<=min(g,100) and (p=='?' or p==3*w+d): break
            print(g, w, d, g-w-d, 3*w+d)

        else:
            if w != '?' and d != '?': pass
            elif w != '?': d = p-3*w if p != '?' else g-w-l
            elif d != '?': w = (p-d)//3 if p != '?' else g-d-l
            else:
                for wd in range(10404):
                    w, d = wd//102, wd%102
                    if w+d+l==g and (p=='?' or p==3*w+d): break
            print(g, w, d, l, 3*w+d)

Solution Explanation

Loop through each line and save each field as variable $g, w, d, l$, and $p$ for games played, won, drew, lost, and points. Initialize an unknown variable to 5, which keeps track of how many unknowns we have in our fields. We decrement this as we check our variables are not missing ('?'). If no unknown, print out the line as is.

Case 1: games and losses unknown.

  • If $w$ and $d$ are known, nothing extra to compute, so pass.
  • If $w$ is known, we can deduce $d$ from $p=3w+d$ or $d=p-3w$
  • If $d$ is known, we can deduce $w$ from $p=3w+d$ or $w=(p-d)/3$
  • If both $w$ and $d$ are unknown, we brute force over pairs of $(w,d)$. Trying $w\in [0,101]$ and $d\in [0,101]$ using a loop over $102\times 102=10404$. We add the constrain that $w+d< 101$, forcing $w+d\leq 100$ and $p=3w+d$
  • After determining $(w,d)$, we print the max games $g=100$ and $l=0$.

Case 2: games unknown and losses known.

  • If $w$ and $d$ are known, nothing extra to compute, so pass.
  • if $w$ is knwon, we can deduce $d$ from $p=3w+d$ or $d=p-3w$
  • If $d$ is known, we can deduce $w$ from $p=3w+d$ or $w=(p-d)/3$
  • We similarly brute force over pairs of $(w,d)$, but now under the constraint $w+d+l< 101$, forcing $w+d+l\leq 100$ (since we know $l$). If $p$ is known, enforce $p=3w+d$.
  • After determining $(w,d)$, set games to $g=w+d+l$, and $p=3w+d$.

Case 3: games known and losses unknown.

  • If $w$ and $d$ are known, nothing extra to compute, so pass.
  • if $w$ is knwon, we can deduce $d$ from $p=3w+d$ or $d=p-3w$
  • If $d$ is known, we can deduce $w$ from $p=3w+d$ or $w=(p-d)/3$
  • We similarly brute force over pairs of $(w,d)$, but now under the constraint $w+d\leq g$, and since $g\leq 100$, we take the minimum of the two. If $p$ is known, enforce $p=3w+d$.
  • After determining $(w,d)$, set losses to $l=g-w-d$ and points to $p=3w+d$.

Case 4: games and losses known.

  • If $w$ and $d$ are known, nothing extra to compute, so pass.
  • if $w$ is knwon, we can deduce $d$ from $p=3w+d$ or $d=p-3w$
  • If $d$ is known, we can deduce $w$ from $p=3w+d$ or $w=(p-d)/3$
  • We similarly brute force over pairs of $(w,d)$, but now under the constraint $w+d+l=g$. If $p$ is known, enforce $p=3w+d$.
  • After determining $(w,d)$, set points to $p=3w+d$.
