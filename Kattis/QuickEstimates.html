Quick Estimates

Problem Statement

You are given a list of numbers are you want to print out the number of digits in each number.

Input

  • First number is an integer $N$.
  • The next $N$ lines are various digits of orders of magnitude.

Output

The ordering of the temperature sequence that satisfies the requirement.

Reference Code (Python)


n = int(input())

for i in range(n):
    s = input().strip()
    print(len(s))

Solution Explanation

Trivial, you go through each number, and print out the length of the string.
