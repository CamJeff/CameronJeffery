Dominoes 2

Problem Statement

Given a set of dominoes that are knocked down by hand, determine the total number of dominoes that fall.

Input

  • One number, specifying the number of test cases that follow.
  • Three integers $n,m,\ell$, where $n$ is the number of dominoes, numbered 1 to $n$.
  • $m$ lines that contain two integers $x$ and $y$, where if domino $x$ falls, it will cause domino $y$ to fall as well.
  • $\ell$ lines that contain a single integer $z$ indicating the domino numbered $z$ is knocked over by hand.

Output

The total number of dominoes that fall.

Reference Code (Python)


def main():
    t = int(input())
    for _ in range(t):
        n, m, l = map(int, input().split())
        g = [[] for _ in range(n + 1)]
        for _ in range(m):
            x, y = map(int, input().split())
            g[x].append(y)

        visited = [False] * (n + 1)
        fallen = 0

        for _ in range(l):
            z = int(input())
            if not visited[z]:
                stack = [z]
                visited[z] = True
                fallen += 1
                while stack:
                    v = stack.pop()
                    for u in g[v]:
                        if not visited[u]:
                            visited[u] = True
                            fallen += 1
                            stack.append(u)

        print(fallen)

if __name__ == "__main__":
    main()

Solution Explanation

The basic idea is to treat the dominoes that knock each other over as a directed graph and for each hand-knocked domino $z$, run a DFS to mark all nodes reachable from $z$. Use a visited array so that we never count a domino twice. The answer is the number of unique visited dominoes.

Number of test cases saved as t. Save variables $n$, $m$, and $\ell$. Create an adjacency list for $n$ dominoes or vertices in our graph. For the pairs of dominoes interactions in the $m$ lines that follow, save them in the adjacency list. Create a visited array for for all dominoes, initializing all of them to False. Set up a fallen counter for fallen dominoes

Iterate through the $\ell$ lines of dominoes we knock over by hand. Save each domino in variable $z$. If we haven't visited this domino, push it into a stack, and mark visited to True, and increment fallen.

While our stack is not empty, pop an element, and go through each of its neighbours in the adjacency list. If we haven't visited a neighbour, mark visited to True, increment fallen, and push to stack.
