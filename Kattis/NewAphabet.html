A New Alphabet

Problem Statement

You are given a guide to how each English alphabet letter maps to a new character string. For any input English sentence, translate it to a new sentence using the mapping given.

Input

An English sentence.

Output

The translated sentence

Reference Code (Python)


alpha_map = {
    "A": "@", "B": "8", "C": "(", "D": "|)", "E": "3",
    "F": "#", "G": "6", "H": "[-]", "I": "|", "J": "_|",
    "K": "|<", "L": "1", "M": "[]\\/[]", "N": "[]\\[]",
    "O": "0", "P": "|D", "Q": "(,)", "R": "|Z", "S": "$",
    "T": "'][\'", "U": "|_|", "V": "\\/", "W": "\\/\\/",
    "X": "}{", "Y": "`/", "Z": "2"
}

def translate_char(char):
    if "A" <= char.upper() <= "Z":
        return alpha_map[char.upper()]
    else:
        return char

starting_string = input()
new_string = ""
for char in starting_string:
    new_string += translate_char(char)
    
print(new_string)

Solution Explanation

Create a dictionary that shows how each letter maps to a new character sequence. Create a function that translates each character (it must be a letter). Create an empty string and add each translated letter as we loop through our input string.
