Birds on a Wire

Problem Statement

A wire of $\ell$ cm is a place where birds can sit, but they can only sit a distance $d$ apart. They also can't sit within 6 cm of either end of the wire. Some birds may already be positioned on the wire. We want to find out the maximum number of birds allowed on wire.

Input

  • length of wire $\ell$, spacing between birds $d$, and number of birds sitting on wire $n$.
  • Next lines are positions of birds already on wire.

Output

The maximum number of birds allowed on wire.

Reference Code (Python)


def main():
    L, d, n = map(int, input().split())
    birds = sorted(int(input()) for _ in range(n))

    if n == 0:
        print(0 if L < 12 else ( (L - 12) // d + 1 ))
        return

    ans  = (birds[0] - 6) // d
    for i in range(1, n):
        ans += max((birds[i] - birds[i-1]) // d - 1, 0)
    ans += ((L - 6) - birds[-1]) // d

    print(ans)

if __name__ == "__main__":
    main()

Solution Explanation

Save information in variables L, d, and n. Sort the birds in ascending order of position. Start with case with no birds on wire yet. This would just be the length of wire (with 6 cm off each end trimmed) divided by the distance between each bird, rounded down.

With variable ans, start with left side from 6 cm to first bird, and place as many as we can. Then for each bird in between, we loop through and floor divide by consecutive birds. We take 0 if there is no space. Finally, we increment ans by taking the right end of wire (minus 6 cm) and the right most bird and determine how much space there is again and how many birds can fit.
