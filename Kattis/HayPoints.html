Hay Points

Problem Statement

You are given a list of words with a salary number next to them. Then, you are given some job descriptions that may contain some of these words listed above. For each description, sum up the salary numbers associated to the words that are contained in these job descriptions.

Input

  • Two numbers, one for each word and its salary number, and the other for the number of job descriptions.
  • Some words with their salary numbers separated by a space.
  • Some job descriptions, each separated by a line containing a period.

Output

The sum of the salary numbers associated to words that are contained within the job descritpion.

Reference Code (Python)


from collections import defaultdict

word_limit, job_limit = list(map(int, input().split()))

dictionary = defaultdict(lambda: 0)
for i in range(word_limit):
    word, salary = input().split()
    dictionary[word] = int(salary)

jobs_shown = 0
hay_points = 0
while jobs_shown < job_limit:
    sentence = input()

    if sentence != ".":
        words = sentence.split()
        for word in words:
            hay_points += dictionary[word]
    else:
        jobs_shown += 1
        print(hay_points)

        hay_points = 0

Solution Explanation

First save the number of words and the number of job descriptions in two variables.

Initialize an empty dictionary. Then, loop through the number of words and store the word name as the key and the salary number as the value in the dictionary.

Initialize the hay points to zero, and the number of descriptions seen to zero. Split all words in the description in an array, and iterate through the array and increment hay points by the salary number of any words that match in our dictionary. Once our sentence is complete, increment our job number, print our hay points, and set it to zero for the next job description.
