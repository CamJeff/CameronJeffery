Where's My Internet?

Problem Statement

You are given houses numbered $1$ to $N$. House $1$ has internet, and every house that is connected to a house with internet will have internet. Given a list of pairs of houses that indicate that they are connected, print out the house numbers with no internet. If all houses have internet, print out Connected.

Input

  • Two numbers, $N$ number of houses, and $M$ number of network cables
  • The next $M$ lines are pairs of houses that are connected by a network cable

Output

The houses, in increasing order, that don't have internet. If all do, print Connected.

Reference Code (Python)


import sys
def main():
    houses, _ = [int(x) for x in input().split()]
    connections = [[] for _ in range(houses+1)]
    for x, y in [[int(y) for y in x.split()] for x in sys.stdin]:
        connections[x].append(y)
        connections[y].append(x)
    connected = [False for _ in range(houses+1)]
    connected[1] = True
    to_check = list(connections[1])
    while to_check:
        t = to_check.pop()
        if not connected[t]:
            connected[t] = True
            for t2 in connections[t]:
                to_check.append(t2)
    unconnected = [x for x in range(2, houses+1) if not connected[x]]
    if not unconnected:
        print('Connected')
    else:
        for x in unconnected:
            print(x)
main()

Solution Explanation

We build an undirected graph for each house, and run a depth first graph traversal from house 1, marking every reachable house as "connected" via a Boolean array. We then collect all houses/vertices not reached by the traversal, and print them in increasing order.

First save the number of houses in variable houses, and the number of connected in an underscore variable that we don't use. Do this by list comprehension to convert to integers.

Build an adjacency list called connections sized at houses+1 just so that indices match the house numbers (there is no house 0). Then read all connection pairs in our input, and add them to our adjacency list, and since graph is undirected, you add them both ways

Create a Boolean array called connected that will indicate whether each house has internet. Initialize all to False to start with. Then, update the first house to True, since we know it has internet.

Start a stack called to_check that contains all immendiate neighbours of house $1$, by using the connections adjacency list we have. While the stakc is not empty, we pop an element. If this element is not connected to internet, update it to True. Then, iterate through all neighbours of this element/house, and push them onto the to_check stack.

Then, in an unconnected list, add every house (start checking from house 2) that is not marked False in the connected array. These houses don't have internet. Then, simply, print out these houses (if there are any).
